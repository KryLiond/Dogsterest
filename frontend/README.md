# React + TypeScript + Vite

Запуск проекта:

1. Запускаем backend:

- npm i
- npm run dev
  И видим сообщение Server is running on http://localhost:4200, бэк запущен.

2. Запускаем frontend:

- npm i
- npm run dev
  Переходим по ссылке, сформированной сборщиком vite, фронт запущен.
